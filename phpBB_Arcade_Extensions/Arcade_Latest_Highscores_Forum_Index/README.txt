# phpBB Arcade - Arcade latest highscores on forum index

## Quick Install
You can install this on the latest release of phpBB 3.3. by following the steps below:

1. Unzip the downloaded release.
2. Copy the 'arcade_latesthighscores' directory in the  (your forum root)/ext/jv/) directory.
3. Navigate in the ACP to 'Customise -> Manage extensions'.
4. Look for 'phpBB Arcade - Arcade latest highscores on forum index' under the Disabled Extensions list, and click its 'Enable' link.
5. Set up and configure 'phpBB Arcade - Arcade latest highscores on forum index' by navigating in the ACP to 'Arcade/Extensions' -> 'phpBB Arcade - Arcade latest highscores on forum index'.

## Update

1. Navigate in the ACP to 'Customise -> Manage extensions' - Look for 'phpBB Arcade - Arcade latest highscores on forum index' under the Enabled Extensions list, and click its 'Disable' link.
2. Unzip the downloaded release.
3. Copy the 'arcade_latesthighscores' directory in the  (your forum root)/ext/jv/) directory.
4. Navigate in the ACP to 'Customise -> Manage extensions' - Look for 'phpBB Arcade - Arcade latest highscores on forum index' under the Disabled Extensions list, and click its 'Enable' link.
5. Set up and configure 'phpBB Arcade - Arcade latest highscores on forum index' by navigating in the ACP to 'Arcade/Extensions' -> 'phpBB Arcade - Arcade latest highscores on forum index'.

## Uninstall

1. Navigate in the ACP to 'Customise -> Extension Management -> Extensions'.
2. Look for 'phpBB Arcade - Arcade latest highscores on forum index' under the Enabled Extensions list, and click its 'Disable' link.
3. To permanently uninstall, click 'Delete Data' and then delete the '/ext/jv/arcade_latesthighscores' directory from your server via FTP.
